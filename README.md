# 个人博客系统（SSM）

## 项目简介
本项目是一个基于 **Spring + SpringMVC + MyBatis（SSM）** 的个人博客系统，实现了用户登录、文章管理和评论功能，是一个典型的 Web 后端项目。

## 技术栈
- 后端框架：Spring、SpringMVC、MyBatis（SSM）
- 数据库：MySQL
- 构建工具：Maven
- 开发环境：IntelliJ IDEA + JDK 8
- 版本管理：Git + GitHub

## 主要功能
- 用户登录/注册  
- 文章发布、编辑、删除、查询  
- 文章列表展示  
- 文章详情页  
- 用户评论功能  
- 登录拦截（未登录不能操作文章）

## 项目架构
采用三层架构：
- Controller 层：处理请求
- Service 层：业务逻辑
- Mapper 层：与数据库交互

## 运行说明
1. 导入 MySQL 数据库（根据 jdbc.properties 配置）  
2. 使用 IDEA 导入 Maven 项目  
3. 部署到 Tomcat 运行
