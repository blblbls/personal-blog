# Personal Blog System（个人博客系统）

## 📌 项目简介
本项目是基于 **SSM（Spring + SpringMVC + MyBatis）** 架构开发的个人博客管理系统，支持用户登录、文章管理、评论管理等核心功能，适合作为学习 Java Web 开发和后端开发的综合实践项目。

## 🛠 技术栈
- 后端框架：Spring + SpringMVC + MyBatis（SSM）
- 数据库：MySQL
- 构建工具：Maven
- 开发环境：IntelliJ IDEA + Git + GitHub
- 前端：JSP + HTML + CSS + JavaScript

## ✨ 核心功能
- ✅ 用户登录/注销  
- ✅ 文章发布、编辑、删除、查询  
- ✅ 文章分页展示  
- ✅ 用户评论功能  
- ✅ 基于拦截器的登录校验  
- ✅ 全局异常处理  

## 📂 项目结构
src/
├── controller # 控制层
├── service # 业务层
├── mapper # MyBatis 持久层
├── pojo # 实体类
└── resources # 配置文件

## 🚀 运行说明
1. 导入项目到 IntelliJ IDEA  
2. 配置 MySQL 数据库并修改 `jdbc.properties`  
3. 使用 Tomcat 部署运行  
4. 访问：http://localhost:8080/
